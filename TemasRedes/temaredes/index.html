10 Conceitos de Redes

O que é uma rede:

Um conjunto de dois ou mais dispositivos conectados que podem se comunicar entre si.

Diferença entre LAN, MAN e WAN:

Escopo é o tamanho ou alcance geográfico da rede. LAN: Local Area Network; Rede privada ou interna para uma escola, empresa ou casa. WAN: Wide Area Network; Rede de área geográfica muito maior, a Internet é um exemplo de rede WAN, ela é um aglomerado de WAN. MAN: Metropolitan Area Network; alguns exemplos são redes de telefones ou TV a cabo, tem uma área geográfica mais ou menos do tamanho de uma metrópole.

Modelo OSI x TCP/IP:

OSI: Modelo de referência pro projeto de protocolos de redes. Mostra como deve ser o protocolo de redes TCP/IP: É uma implementação baseada no modelo OSI e é empregada em redes locais.Pilha de protocolos com as camadas modificadas

Protocolos de Rede:

Protocolo é uma “documentação”, um padrão que é composta de regras, procedimentos e especificações para formatos de dados, esses protocolos indicam como deve ser feita a comunicação entre dispositivos dentro de uma rede, tem a forma de softwares.

Endereço IP:

Realizar o endereçamento IP (Internet Protocol) serve para fazer uma identificação lógica dos hosts (host é todo dispositivo que está conectado em uma rede) dentro de uma red

Máscara de Subrede:

É uma máscara feita de um conjunto de bits que é aplicada sobre o endereço IP, que resulta no endereço de rede

Diferença entre roteador e switch:

Roteador: Dispositivo usado para unir redes distintas, efetua o encaminhamento de pacotes de dados entre redes de computadores distintas. Switch: Utilizado para interligar os hosts dentro de um rede

RFCs: Request for Comments.

documento de rede que documenta padrões, serviços e protocolos para redes locais ou na Internet. São milhares de documentos muito técnicos.

Método de transmissão:

Meio físico que suporta a propagação de sinais para as telecomunicações, entre dispositivos. Cabos UTP/STP(Par Trançado), Cabo coaxial, Wireless (ondas eletromagnéticas), Fibras Ópticas

Conceitos de segurança de redes:

  • Entender Firewall: Dispositivos de segurança;
  • VPN: Comunicação segura;
  • IDS/IPS;
  • Criptografia;
  • Permissões e Controle de Acesso;
  • Malware e software anti Malware;
  • Ferramentas(Sniffers de Rede, Teste de Penetração, Scanners de Vulnerabilidade, etc).
