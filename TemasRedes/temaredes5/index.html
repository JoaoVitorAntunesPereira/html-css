O que é uma Rede

Um conjunto de dois ou mais dispositivos conectados que podem se comunicar entre si.

Diferença entre LAN, MAN e WAN

A principal diferença entre LAN, MAN e WAN é seu escopo, o tamanho ou alcance geográfico da rede

Modelo OSI x TCP/IP

OSI: Modelo de referência pro projeto de protocolos de redes.TCP/IP: É uma implementação baseada no modelo OSI e é empregada em redes locais.

Protocolos de Rede

Protocolos são um documento de padronização, indicam como deve ser feita a comunicação entre dispositivos dentro de uma rede.

Endereço IP

IP (Internet Protocol) serve para fazer uma identificação lógica dos hosts dentro de uma rede.

Máscara de Subrede

É uma máscara feita de um conjunto de bits que é aplicada sobre o endereço IP, que resulta no endereço de rede.

Diferença entre roteador e switch

Roteador: Dispositivo usado para unir redes distintas. Switch: Utilizado para interligar os hosts dentro de um rede.

RFCs: Request for Comments

Documento de rede que documenta padrões, serviços e protocolos para redes locais ou na Internet.

Método de transmissão

Meio físico que suporta a propagação de sinais para as telecomunicações, entre dispositivos.
